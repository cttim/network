    In this example, the client will send a sentence to the server. The server capitalizes the sentence and sends it back to the client.
    -------------------
    How to compile:
        javac UDPClient.java
        javac UDPServer.java
    -------------------------------
    How to run: (in this code we assume that both the client and server are runnning on the same machine)
    run first the server:
        java UDPServer
    Then in another terminal run the client:
        java UDPClient
